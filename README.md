# Avnac

Avnac is an open canvas for layouts, posters, and graphics.

## Tech Stack

### Frontend
- **React** + **Vite**
- **Fabric.js** for canvas manipulation and graphic rendering
- **Tailwind CSS** for styling
- **TanStack Router** for client-side routing
- **Motion** for animations
- **Zod** for schema validation


### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

## Features

- **Creative Canvas**: Build distinct layouts, posters, and graphics.
- **Autosave**: Files automatically save in the browser.
- **Exporting**: PNG export is available directly in the editor.
